주의사항

※ 「마누카」의 가슴 쉐이프키는 독특한 형태로 되어 있습니다.
기본 가슴에서 크기를 적당히 키우는 "Breast_big" 쉐이프키가 존재하며,
이 상태에서 크기를 더 키울수 있는 "Breast_big_PLUS" 쉐이프키가 존재합니다.

"Breast_big_PLUS" 쉐이프키는 단독으로 사용하는 것이 아닌,
"Breast_big" 쉐이프키의 수치가 최대로 적용된 상태에서 추가로 사용하는 형식입니다.
이러한 방식은 중간 수치의 가슴 모양을 예쁘게 하기 위해 사용하였습니다.

"Breast_big_PLUS" 까지 크기를 키울 경우에는 기본 의상의 모양이 부자연스럽게 될 수 있습니다.

기본 의상은 속옷 메쉬를 OFF 한 상태로 착용하는 것을 전제로 제작되었습니다.
속옷의 메쉬가 OFF 되어 있기 때문에, 개변 시 필요할 때 사용해주시길 바랍니다.

도입 방법

Unity 2019.4.31f1와 VCC를 설치 후, VCC에서 Avatar 새 프로젝트를 생성하고 Open Project를 클릭합니다.
※2023년 8월 31일 기준의 Unity 버전이며, 이후 VRChat의 업데이트로 인해 사용하는 Unity 버전이 달라질 수 있습니다.

이후 위 링크에서 원하는 쉐이더를 다운로드 한 후 Unitypackage를 프로젝트에 import 하고, 아바타의 Unitypackage도 import합니다.
Assets - (아바타명) - Prefab 폴더에서 원하는 Prefab을 선택 후, Hierarchy에 드래그 앤 드롭합니다.
상단 메뉴의 VRChat SDK - Show Control Panel - Builder - Build & Publish for Windows 순으로 클릭 후 아바타를 업로드합니다.
