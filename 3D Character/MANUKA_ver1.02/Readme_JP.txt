注意事項

※ 「マヌカ」の胸シェイプキーはユニークな形になっています。
基本胸からサイズを適度に大きくする"Breast_big"シェイプキーが存在し、
この状態でサイズをさらに大きくできる"Breast_big_PLUS"シェイプキーが存在します。

"Breast_big_PLUS" シェイプキーは単独で使用するのではなく、
"Breast_big"シェイプキーの数値が最大に適用された状態で追加で使用する形式です。
このような方式は、中間数値の胸の形をきれいにするために使用しました。

"Breast_big_PLUS"までサイズを大きくする場合は、基本衣装の形が不自然になることがあります。

基本衣装は下着メッシュをOFFの状態で着用することを前提に製作されました。
肌着のメッシュがOFFになっていますので、改変時に必要なときにご使用ください。

導入方法

Unity 2019.4.31f1とVCCをインストールした後、VCCでAvatarの新しいプロジェクトを作成し、Open Projectをクリックします。
※2023年8月31日基準のUnityバージョンであり、その後VRChatのアップデートにより使用するUnityバージョンが変わることがあります。

その後、上記リンクからご希望のシェーダーをダウンロードした後、Unitypackageをプロジェクトにインポートし、アバターのUnitypackageもインポートします。
Assets - (アバター名) - Prefabフォルダから希望するPrefabを選択した後、Hierarchyにドラッグアンドドロップします。
上段メニューのVRChat SDK - Show Control Panel - Builder - Build & Publish for Windowsの順にクリックし、アバターをアップロードします。
